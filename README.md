# Spotify Clone 🎵

A simple Spotify Clone built using HTML, CSS, and JavaScript.  
This project recreates the basic UI and music player functionality of Spotify.

---

## 🚀 Features

- 🎶 Play/Pause music
- ⏭ Next and Previous song controls
- 📀 Dynamic song list
- 🎧 Responsive music player UI
- 🔊 Progress bar control
- 📱 Responsive design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
